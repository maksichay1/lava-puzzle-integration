# Интеграция Puzzle Bot + Lava TOP

Развертывание на Railway.com

## Быстрый старт:

1. Нажмите кнопку "Deploy on Railway"
2. Настройте переменные окружения
3. Получите ваш домен
4. Настройте кнопки в Puzzle Bot

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template?template=https://github.com/yourusername/lava-puzzle-integration)
